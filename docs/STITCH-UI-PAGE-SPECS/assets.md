# 素材中心

## 1. 頁面用途

- 管理素材包與版本；左側列表、右側詳情與多個對話框。
- 主要給設計與營運。

## 2. 進入方式

- route：`/assets`。
- 側欄更多 → 素材中心。

## 3. 首屏必須看到的內容

1. SidebarTrigger 與標題「素材中心」。
2. 左欄「新增素材包」按鈕。
3. 左欄列表或載入／錯誤／空狀態。
4. 右欄 AssetsRightPanel。

## 4. 區塊排列順序

- Header 全寬；下方左 380px 列表加右側內容；對話框疊加。

## 5. 按鈕盤點

### A. 主按鈕（Primary CTA）

- 新增素材包。

### B. 次按鈕（Secondary）

- 右側表單動作待確認。

### C. 危險按鈕（Danger）

- 待確認對話框內。

### D. 輔助按鈕（Utility）

- 列表點選。

## 6. 篩選 / 控制元件

- 待確認 useAssetsWorkbench 是否含搜尋。

## 7. 狀態資訊 / badge / warning

- 載入中、載入失敗、尚無素材包。

## 8. 可收合但不能消失的區塊

- 左右雙欄不可改為單欄精靈而無列表。

## 9. 此頁最不能被 AI 誤改的地方

- 新增入口；錯誤文案；選取高亮。

## 10. Stitch 用摘要

素材中心為 master-detail：左列表右詳情。首屏左側要有新增與包列表。主 CTA 新增素材包。桌面版勿移除列表欄；錯誤與空狀態須清楚。
