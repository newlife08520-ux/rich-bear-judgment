# 素材工廠效率與命中率

## 1. 頁面用途

- 輔助儀表：KPI 定義、工廠效率區塊 A、團隊命中率區塊 B（按商品或按人）、知識回流區塊 C。非首頁主決策表。
- 主要給老闆與管理。

## 2. 進入方式

- route：`/scorecard`。
- 側欄更多 → 素材工廠效率與命中率。

## 3. 首屏必須看到的內容

1. 完整頁標題與 BarChart3。
2. 灰底定位說明與 KPI 條列、琥珀色 0% 提醒。
3. 區塊 A 素材工廠效率標題與待 API 說明。
4. 區塊 B 團隊命中率與兩個 tab。

## 4. 區塊排列順序

- Header → 定位卡 → A 卡 → B 卡含 Tabs 與表格 → C 卡。

## 5. 按鈕盤點

### A. 主按鈕（Primary CTA）

- 無；主互動為 TabsTrigger 按商品／按人。

### B. 次按鈕（Secondary）

- 同上 tab。

### C. 危險按鈕（Danger）

- 無。

### D. 輔助按鈕（Utility）

- SidebarTrigger。

## 6. 篩選 / 控制元件

- groupBy 兩態；無頁級日期選擇器於本檔。

## 7. 狀態資訊 / badge / warning

- 琥珀提醒；載入與尚無資料；A 與 C 待 API 文案需保留。

## 8. 可收合但不能消失的區塊

- 定位與 KPI 卡；淘汰原因列表於各項卡片內。

## 9. 此頁最不能被 AI 誤改的地方

- 非主表聲明；商品／人雙視角；待 API 免誤解。

## 10. Stitch 用摘要

成績單頁以說明卡開場再進三區塊。B 區用 tab 切換視角。無單一主按鈕。設計勿刪待 API 提示；應利於長說明與多卡片表格並存。
