# 使用流程重整方案（投手視角）

本文件為「素材中心 × 投放中心」使用流程重整的具體方案，不涉及直接改程式，先供確認方向後再實作。

---

## 一、素材中心：重新定義

**定位**：只做「管理素材」與「預設創意模板」。目標是讓人**先快速把素材丟進去、分好版本，再慢慢補文案**。

### 1. 素材中心第一屏保留哪些欄位

| 欄位 | 說明 | 必填/選填 |
|------|------|-----------|
| **素材包名稱** | 辨識用，例如「春季主圖組」 | 必填 |
| **產品名稱** | 取代現有「品牌/產品」，一個欄位即可 | 必填 |
| **預設文案** | 主文案，可後補 | 選填 |
| **預設標題** | 可後補 | 選填 |
| **預設 CTA** | 可後補，預設可給「來去逛逛」 | 選填 |
| **預設網址** | 落地頁 URL，可後補 | 選填 |
| **說明** | 備註 | 選填 |
| **素材版本** | 影片/圖片、比例、上傳（或貼網址）、檔名、是否主版本 | 必填至少一版 |

第一屏的**順序建議**：  
名稱 → 產品名稱 → 立刻進入「素材版本」區（上傳/新增版本）→ 下面才是預設文案、標題、CTA、網址、說明。  
也就是「先能丟檔案、選類型與比例」，文案區可以摺疊或放在下方，標註「可後補」。

### 2. 素材中心哪些欄位改成進階設定或拿掉

| 現有欄位 | 處理方式 | 說明 |
|----------|----------|------|
| **廣告目的**（sales/leads/traffic/engagement） | **拿掉或收進「進階」** | 不卡主流程；若後端仍需欄位，可預設「銷售」並在進階裡改 |
| **狀態**（draft/ready/archived） | **拿掉或收進進階** | 第一版可不顯示，或僅在列表小字顯示，建立時不問 |
| **品牌/產品名稱** | **改名為「產品名稱」、合併為一欄** | 不再用「品牌/產品」這種正式雙重欄位 |
| **主文案 / 標題 / CTA / 網址** | **保留但標註「可後補」、不擋建立** | 建立素材包時可留空，儲存後再補；不要必填 |

**不卡住上傳流程**的具體做法：

- 建立「素材包」時：必填只有「素材包名稱」「產品名稱」。
- 新增「素材版本」時：必填只有「類型（影片/圖片）」「比例」與檔案（上傳或 URL）。檔名可自動帶入或選填。
- 其餘（主文案、標題、CTA、網址、說明、廣告目的、狀態）一律不擋儲存，可事後在編輯時補。

---

## 二、投放中心：重新定義

**定位**：只用現成素材「建立投放」，不是重做一次素材單。流程要短、選項要少。

### 3. 投放中心主流程改成哪三步

| 步驟 | 名稱 | 內容 |
|------|------|------|
| **Step 1** | **選投放起點** | 二選一：**從頭建立** 或 **從上一組複製（快速變體）**。若選「從上一組複製」，先選一筆既有草稿，系統帶入模板（見第六點）。 |
| **Step 2** | **選素材包** | 下拉選一個素材包。選完後文案/標題/CTA/網址**預設全部沿用素材包**，畫面上不展開文案編輯區。 |
| **Step 3** | **選素材版本 + 投放設定** | 勾選要用的素材版本（可多選）；同一區塊內只顯示**真正投放要的欄位**：廣告帳號、粉專/IG、預算、受眾、placement、命名、排程（選填）。 |

也就是：**起點（新建 or 複製）→ 素材包 → 素材版本與投放設定**。不再出現「一大張表單從頭填到尾」。

### 4. 哪些欄位預設沿用素材包

以下欄位**一律預設沿用**所選素材包的對應值，投放中心不預設展開編輯：

- 主文案  
- 標題  
- CTA  
- 落地頁網址  
- 說明/備註  

實作上：選完素材包後，表單只顯示「目前沿用：素材包 OOO 的文案/標題/CTA/網址」的摘要（一行或小卡），不顯示五個輸入框。

### 5. 哪些欄位只有在覆寫時才展開

當使用者明確點「我要改文案」或「覆寫創意」時，才展開：

- 主文案  
- 標題  
- CTA  
- 說明  
- 落地頁網址  

也就是：**預設摺疊，按「覆寫文案」再展開**。展開後可選擇性填，有填的才覆寫，沒填的仍沿用素材包。

**廣告目標**：預設就是「轉換」，畫面上**不顯示**流量/名單/互動等選項（或收到「進階」裡）。除非之後有明確需求，否則主流程只看到「轉換」或甚至不顯示該欄位（後端固定送轉換）。

## 三、複製功能改成「快速變體」

### 6. 複製功能怎麼改成快速變體

目前「複製」= 複製一筆草稿資料、開表單預填。你要的是**投手用的快速變體**：

| 行為 | 說明 |
|------|------|
| **入口** | 投放中心列表或 Step 1 提供「從上一組複製」或「複製成變體」。 |
| **選來源** | 選一筆既有草稿當模板。 |
| **自動帶入** | 帳號、粉專/IG、預算、受眾、placement、排程、**素材包**、**命名規則** 全部沿用。 |
| **自動命名** | 名稱自動加後綴，例如「原名稱 A」「原名稱 B」或「原名稱 01」「原名稱 02」，可設定偏好（A/B/C 或 01/02/03）。 |
| **只換素材版本** | 預設選同一素材包，但**清空/重選素材版本**，讓人只勾選不同尺寸或不同檔。 |
| **文案** | 預設沿用素材包，若有需要再點「覆寫文案」微調。 |

也就是：**複製 = 同一套投放設定 + 同一素材包 + 換版本 + 自動改名**，而不是單純複製一筆資料再手動改一堆欄位。

## 四、UI 精簡：刪掉或延後

### 7. 哪些現在 UI 可以直接刪掉或延後

| 項目 | 建議 |
|------|------|
| 素材中心：廣告目的選單 | 主流程拿掉；若後端必填則預設「銷售」並在進階改。 |
| 素材中心：狀態選單 | 主流程拿掉或僅列表顯示；建立時不問。 |
| 素材中心：品牌/產品分開或過於正式 | 合併為一個「產品名稱」欄位。 |
| 投放中心：Campaign 目標選單（流量/名單/互動/轉換） | 主流程不顯示；固定為「轉換」或收進進階。 |
| 投放中心：文案/標題/CTA/網址/說明 五個輸入框常駐 | 改為預設不展開，只顯示「沿用素材包」摘要；按「覆寫文案」才展開。 |
| 投放中心：粉專/IG 等 ID 若很少用 | 可收進「進階」或同一區塊但視覺弱化。 |
| 投放中心：排程開始/結束 | 維持選填，可放在同一區塊下方或進階。 |
| 狀態（草稿/待發佈/已發佈/失敗） | 列表保留；表單內可弱化或固定預設「草稿」。 |

**原則**：主流程只保留「選起點 → 選素材包 → 選版本 + 填預算/受眾/placement/命名」；其餘一律後補或進階，減少畫面噪音。

## 五、總結對照

| 主題 | 現狀痛點 | 重整後 |
|------|----------|--------|
| 素材中心 | 重複填文案、廣告目的/狀態卡流程 | 第一屏：名稱、產品名稱、素材版本；文案可後補；廣告目的/狀態拿掉或進階 |
| 投放中心 | 像重做素材單、欄位太多 | 三步：起點 → 素材包 → 版本+投放設定；文案預設沿用、覆寫才展開 |
| 廣告目標 | 一直看到流量/名單/互動 | 固定轉換，其他收起來或進階 |
| 複製 | 只複製一筆資料 | 快速變體：沿用設定+素材包、自動改名、只換版本、必要時再覆寫文案 |

---

請先確認這份方案是否符合你「站在投手角度」的預期，再進入實作階段。若需要調整哪幾點，直接標出即可。
