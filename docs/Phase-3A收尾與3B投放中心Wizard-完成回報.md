## 4. 驗收步驟

### Phase 3A 收尾

1. **只看我負責**：勾選「只看我負責」→ 確認顯示目前篩選之使用者名稱與 id；若左側為模擬身份且與登入帳號不同，確認出現橘字 mock 提示。
2. **批次回饋**：勾選多筆任務 → 批次改狀態或批次指派；若全部成功，toast 顯示「已更新 N 筆」；若部分失敗（可模擬：無效 id），確認顯示成功／失敗筆數與失敗原因摘要。
3. **截止日**：任務有截止日時，已逾期為紅、今日為橘、3 天內為黃，其餘一般色。

### Phase 3B 投放中心 Wizard

1. 點「建立投放草稿」→ 確認步驟列為「1. 基本設定 → 2. 素材與版本 → 3. 投放前檢查」且副標為 Step1 說明。
2. **Step1**：填寫帳號、目標、Campaign／Ad Set／Ad、預算等 → 點「下一步」→ 進入 Step2。
3. **Step2**：選素材包、選素材版本（可選主素材組、批次建組）→ 點「下一步」→ 進入 Step3。
4. **Step3**：確認粉專/IG、投放前檢查列表、覆寫文案與 CTA／落地頁 → 點「建立」完成；可點「上一步」回 Step2。
5. 編輯草稿：開啟後同樣為三步驟，可切換步驟後在 Step3 儲存。
