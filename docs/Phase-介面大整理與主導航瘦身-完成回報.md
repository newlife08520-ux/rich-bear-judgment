# Phase 介面大整理、主導航瘦身、閱讀體驗重構 — 完成回報

## 1. 完成狀態

**本輪介面重構已完成。** 未新增新功能，以「介面大整理、資訊層級重排、主導航瘦身、閱讀體驗」為目標，完成 RICH BEAR 審判官淺色裁決工作台、門檻設定經營語言與百分比、主導航日常／管理分層、任務中心工作台化、資料歸因修正定位調整。

---

## 2. 本輪修改摘要

### 一、RICH BEAR 審判官

- **主閱讀區改淺色**：整頁改為 `bg-gray-50`，主內容區與卡片為白底、`border-gray-200`，不再使用大片深藍／深灰底。
- **輸出與氣泡**：助理回覆改為白底卡片、`prose-gray`、`text-gray-800`，字級與對比提升；使用者氣泡保留品牌 primary 色。
- **決策卡**：改為白底卡片、邊框，標題改為 `text-gray-700`。
- **證據與指標**：右側面板僅保留「證據與指標」，移除「歷史操作」tab；右側改為白底、標題與說明字級提高。
- **歷史操作**：左側歷史改為預設收合，按鈕文案改為「查看更多歷史」／「收合歷史」，不佔主畫面。
- **空狀態**：文案改為「裁決入口 — 拿判決，不是純聊天」。
- **右側／左側面板**：預設皆收合，主區域以閱讀為主；按需展開證據或歷史。

### 二、門檻設定頁

- **百分比顯示與輸入**：`funnelAtcTolerance`、`funnelPurchaseTolerance` 改為 UI 顯示 20%（非 0.2），輸入欄為數字 + 「%」。
- **經營語言**：欄位改為「最低點擊門檻」「最低加購數門檻」「最低購買數門檻」「低花費運氣單判定上限」「加購率容許偏差」「購買率容許偏差」等，並補齊說明。
- **已發布 vs 草稿**：新增「已發布 vs 草稿」表格，並列已發布（使用中）與草稿值，差異列以背景色標示。
- **高風險提示**：停損花費門檻 ≥ 9999、ROI 漏斗最低花費 < 200、規則最低花費 < 200 時顯示高風險提示。
- **三組 preset**：保守測試、標準測試、積極擴量，一鍵套用後可再微調。
- **目前推薦設定說明**：新增卡片，以白話說明三種 preset 適合的投放策略。

### 三、主導航瘦身

- **日常**：主導航僅保留「日常」一組，含今日決策中心、商品作戰室、任務中心、RICH BEAR 審判官、素材中心、素材生命週期、成功率成績單、投放中心、投放紀錄。
- **管理與設定**：商品映射、門檻設定、Prompt 設定、團隊權限、判讀紀錄、FB 帳號分析、GA 頁面分析 移出主導航，改為「設定中心」單一入口；設定中心首頁新增「管理工具」卡片，內含上述連結（資料歸因修正、門檻設定、Prompt、團隊、判讀紀錄、FB、GA）。
- **商品映射**：側欄不再顯示；頁面標題改為「資料歸因修正」，說明改為「此為資料修正工具，非日常主流程」。

### 四、任務中心工作台化

- **三張摘要卡**：進頁先顯示「高優先」「今日到期」「高影響金額」三張卡，數字與副標清楚。
- **預設「我負責」**：`onlyMine` 預設為 `true`，預設視圖為「我負責的任務」。
- **每列操作**：新增「操作」欄，提供前往商品作戰室、素材中心、RICH BEAR 審判官、投放中心之圖鈕。
- **表格**：保留於摘要卡與篩選區下方，非唯一主視圖；到期日與優先級用色維持（紅／橘／黃）。

### 五、整體視覺

- 審判官、門檻設定、任務中心均以淺色為主；深色僅作點綴（如 primary 按鈕、標籤）。
- 文字對比提高（gray-800／gray-700／gray-600），避免「幾乎看不到字」。

## 5. 驗收步驟

1. **主導航**：側欄僅見「日常」（9 項）與「管理與設定」→ 設定中心；進入設定中心可見「管理工具」與 7 個連結。
2. **RICH BEAR 審判官**：整頁淺色、無大片深底；左側歷史預設收合，點「查看更多歷史」展開；右側證據預設收合，點「證據」展開；助理回覆為白底卡片、字清楚。
3. **門檻設定**：容差欄位顯示為 20% 等百分比；已發布 vs 草稿表並列；可套用保守／標準／積極 preset；有推薦設定說明與高風險提示。
4. **任務中心**：進頁先見三張摘要卡；預設為「只看我負責」；每列有操作圖鈕可前往商品／素材／審判／投放。
5. **資料歸因修正**：自設定中心進入，頁面標題為「資料歸因修正」，說明為資料修正工具。
