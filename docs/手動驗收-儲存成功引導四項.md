# 手動驗收：儲存成功後引導卡（四項）

**對應 commit**：74136d1（收尾版，Staging 手動點擊驗收）

**環境**：Staging（或本地 `npm run dev` 後）  
**前置**：已登入，可進入「設定」頁。

---

## 步驟 0：進入設定並觸發「儲存成功」

1. 開啟 **設定** 頁（例如 `/settings`）。
2. 在「設定／驗證／同步」任一分頁做一項可儲存的變更（或保持不變亦可，若後端接受）。
3. 點 **「儲存所有設定」**，確認請求成功（無錯誤提示）。

---

- **預期**：儲存成功後，頁面出現一張引導卡，標題區為「設定已儲存。下一步：」，內文有「立即同步帳號」「前往首頁更新資料」等說明與按鈕。
- **操作**：僅確認該卡有出現即可。
- **通過**：是 / 否

## 驗收 2：點「立即同步帳號」會切到 API 綁定並捲到同步區

- **預期**：點引導卡上的 **「立即同步帳號」** 後，(1) 分頁切換到「API 綁定」，(2) 頁面捲動到「同步帳號」區塊（可見到同步相關按鈕/說明）。
- **操作**：點該按鈕一次，觀察分頁與捲動。
- **通過**：是 / 否
