# 如何開啟 AI 行銷總監

這是**網頁應用**，需要先啟動伺服器，再用瀏覽器開啟。

## 方式一：開發模式（建議）

1. 在專案資料夾開啟終端機（PowerShell 或 CMD）。
2. 執行：
   ```bash
   npm run dev
   ```
3. 看到 `serving on http://localhost:5000` 後，用瀏覽器開啟：
   **http://localhost:5000**

## 方式二：正式版

1. 先建置（若還沒建置過）：
   ```bash
   npm run build
   ```
2. 啟動：
   ```bash
   npm start
   ```
3. 瀏覽器開啟 **http://localhost:5000**

## 若打不開（常見：埠 5000 已被佔用）

- **錯誤訊息出現「EADDRINUSE」或「address already in use」**：代表埠 5000 已被佔用（例如上次開的伺服器沒關）。
  - **解法 A**：關掉其他佔用 5000 的程式。在 PowerShell 查誰在用 5000：
    ```powershell
    netstat -ano | findstr :5000
    ```
    記下最後一欄的 PID，然後關閉該程式或執行 `taskkill /PID 該數字 /F`。
  - **解法 B**：直接改用其他埠啟動，不用關舊程式：
    ```powershell
    $env:PORT=3000; npm run dev
    ```
    然後用瀏覽器開啟 **http://localhost:3000**
- **白畫面／一直載入**：等幾秒讓 Vite 編譯完成，再重新整理頁面。
- **其他錯誤**：把終端機的完整錯誤貼給開發者，方便排查。
